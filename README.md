# Тестовый модуль управления для языка RCML. 

В течение 10 секунд раз в секунду посылает случайное значение по трем осям X,Y,Z.

####Доступные оси
Наименование  | Верхняя граница  | Нижняя граница  | Примечание
------------  | :-----------------:  | :------------------:  | ------------------
X  | 100  | -100  | 
Y  | 1  | 0  | Бинарная
Z  | 100  | 0  |
